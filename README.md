AI-powered chatbot that analyzes the Titanic dataset using:

- FastAPI (Backend)
- LangChain-style agent logic
- Streamlit (Frontend)
- Pandas + Matplotlib visualizations

## Features

- Natural language queries
- Survival analysis
- Gender statistics
- Age histograms
- Class & embarkation analysis

## Run Backend

uvicorn main:app --reload

## Run Frontend

streamlit run app.py
